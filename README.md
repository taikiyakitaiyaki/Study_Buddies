# Study Buddies – Prototype

## Description
Study Buddies is a Python prototype developed for the IS-211 course. The system helps students register courses, find study buddies, and book study rooms with a waiting list.

## Technologies
- Python
- OOP
- Data structures: dictionary, list, deque

## Main classes
- Student
- Courses
- StudyBuddyMatcher
- Room
- RoomManager
  
## Run the program
python main.py

## Purpose
To demonstrate time complexity and use of data structures in a practical application.
